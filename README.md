# Tru-Mi 聖誕企劃 | 故事淬鍊邀請函

這是一個為 Tru-Mi Jewelry 設計的聖誕節行銷一頁式網站，用於推廣「故事淬鍊邀請函」珍藏禮盒。

## 專案結構
- `app.py`: Streamlit 主程式碼，包含所有頁面邏輯與樣式。
- `hero_image.jpg`: 網站首圖，展示高品質的禮盒形象。
- `requirements.txt`: 專案所需的 Python 套件清單。

## 如何執行 (本地端)
1. 確保已安裝 Python。
2. 安裝套件: `pip install -r requirements.txt`
3. 執行網頁: `streamlit run app.py`

## 部署
本專案可直接部署至 Streamlit Community Cloud 或其他支援 Streamlit 的平台。
